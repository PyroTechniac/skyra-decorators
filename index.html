@skyra/decorators

TypeScript decorators for Klasa Projects

About

This package is a collection of useful TypeScript decorators for Klasa projects.

Please note that as of version 2.0.0 this package targets @klasa/core and the new klasa, for the version targeting discord.js see version 1.x!

Contributing

To contribute to this repository, feel free to create a new fork of the repository and submit a pull request. We highly suggest ESLint to be installed in your text editor or IDE of your choice to ensure builds from GitHub Actions do not fail.

  1. Fork, clone, and select the master branch.
  2. Create a new branch in your fork.
  3. Make your changes.
  4. Commit your changes, and push them.
  5. Submit a Pull Request here!
